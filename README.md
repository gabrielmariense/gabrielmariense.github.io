## Arquivos principais para edição

- `js/data.js`: logo, menu, contatos e projetos.
- `js/header.js`: componente padrão do cabeçalho.
- `js/footer.js`: componente padrão do rodapé.
- `js/projects.js`: renderização dos cards de projetos.
- `js/contacts.js`: renderização dos cards de contato.
- `js/script.js`: tema claro/escuro, menu mobile, filtros e validação do formulário.
- `css/style.css`: visual, responsividade e temas.
